## Requirements
* PyTorch >= 1.1.0
* python 3.6
* networkx
* scikit-learn
* scipy
* munkres

## Run from
preset version:
```bash
./scripts/XX.sh
```
or modifying the network parameters and run
```bash
python run.py --param1 xxx --param2 xxx --param3 xxx ...
```
